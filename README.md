# ChronoPlotter

![ChronoPlotter logo](https://github.com/mncoppola/ChronoPlotter/blob/main/images/logo.png?raw=true)

If using an unsigned build for macOS, you will get an error saying the file is 
"damaged".  To be able to run the application, you will need to open a shell (Terminal.app
on macOS) and run the command:

```
cd <directory where ChronoPlotter is installed>
xattr -cr ChronoPlotter.app
```

Easily create clean, customizable charts and graphs of your load development 
sessions with little to no manual data entry. Supports most chronographs and 
e-target systems including Garmin Xero, LabRadar, MagnetoSpeed, Competition Electronics 
ProChrono, and ShotMarker, as well as manual data entry.

This page is only used to view the source code for ChronoPlotter.

**If you'd like to download and use ChronoPlotter, visit here**: https://chronoplotter.com/#download

**For the latest instructions on using ChronoPlotter, visit here**: https://chronoplotter.com/quick-start-guide/
